[사전 환경 구성]

config 폴더 내의 파일 2가지를 작성해주셔야 합니다.

1. OPEN_API_KEY.txt
    - 본인의 OPEN API 인증키를 복사하여 붙여넣고 저장하시면 됩니다.
    (default 값으로 아무 것도 적지 않고 비워 둔 상태입니다.)

2. Fields.txt
    - API 요청으로 받아온 데이터에서 남기고 싶은 필드 값들을 적어주시면 됩니다.

[실제 사용 예시]

    - ipynb 파일 형식으로 예시 코드를 작성해 두었으니 코드를 보시고 참고하시면 됩니다.
    (현재는 get_data_from_api와 get_datas_from_api 2가지 함수를 이용 가능합니다.)
